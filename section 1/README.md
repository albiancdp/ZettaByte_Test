### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/albiancdp/ZettaByte_Test.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Rename .env.example to .env
4. Run project for development
   ```sh
   npm run start:dev
   ```
5. Run project 
   ```sh
   npm run start
   ```
